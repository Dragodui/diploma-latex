\begin{center}
\noindent {{\color{blueZUT}\Large\sffamily  {Streszczenie}}}\\[1cm] 
\end{center}

Przedmiotem niniejszej pracy jest projekt i implementacja systemu informatycznego ,,Household Manager'' wspomagającego zarządzanie gospodarstwem domowym. Aplikacja rozwiązuje problemy związane z podziałem obowiązków, rozliczaniem wspólnych wydatków oraz komunikacją między domownikami. 

System składa się z aplikacji mobilnej stworzonej w technologii React Native oraz serwera backendowego napisanego w języku Go, wykorzystującego bazę danych PostgreSQL. Kluczowe funkcjonalności obejmują: zarządzanie wirtualnymi domami, system przydzielania zadań, moduł finansowy z automatycznym odczytywaniem danych z rachunków (OCR), wspólne listy zakupów oraz ankiety decyzyjne. Zastosowanie nowoczesnych technologii pozwoliło na stworzenie wydajnego, skalowalnego i intuicyjnego w obsłudze rozwiązania, dostępnego na platformy Android i iOS.

\vspace{10pt}
\noindent{\bf słowa kluczowe:} zarządzanie domem, aplikacja mobilna, React Native, Go, OCR, finanse osobiste

\begin{center}
\noindent {{\color{blueZUT}\Large\sffamily {Abstract}}}\\[1cm] 
\end{center}
The subject of this thesis is the design and implementation of the "Household Manager" IT system aimed at supporting household management. The application addresses issues related to the division of chores, settlement of shared expenses, and communication among household members.

The system consists of a mobile application built with React Native technology and a backend server written in the Go language, utilizing a PostgreSQL database. Key functionalities include: management of virtual homes, a task assignment system, a financial module with automatic data extraction from bills (OCR), shared shopping lists, and decision-making polls. The use of modern technologies allowed for the creation of an efficient, scalable, and intuitive solution, available for Android and iOS platforms.

\vspace{10pt}
\noindent{\bf keywords:} household management, mobile application, React Native, Go, OCR, personal finance
